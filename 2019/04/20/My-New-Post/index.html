hexo + github pages 的简单使用

2019-04-20

Typora 书写 .mk（Markdown格式文件）。

命令 说明
npm install hexo –save 安装hexo框架
hexo init xxx 初始化一个项目
cd xxx
hexo server （hexo s） 开启hexo自带服务器
http://localhost:4000
https://github.com 创建仓库 用户名.github.io
hexo clean
hexo generate 生成静态文件于public目录
npm install hexo-deployer-git –save 安装插件 可自动使用git部署静态文件
编辑 _config.yml 文件 deploy:
type: git
repo:
branch: [branch]
hexo deploy 页面部署
https://xxx.github.io 访问页面
